Deskripsi Proyek Deployment Cloud

Proyek ini adalah implementasi dari soal modul deploy website otomatis dari GitHub untuk LKS Cloud Computing tahun 2026. Tujuan utama dari proyek ini adalah mendemonstrasikan kemampuan:

  • Pengelolaan kode sumber menggunkan Git dan GitHub
  • Deployment otomatis statis site dari GitHub ke paltform layanan Cloud (Vercel/Netlify)
  • Penerapan fitur dinamis sederhana (API Publik)
  • implementasi keamanan dasar (.gitignore) dan Logging (console.log)
